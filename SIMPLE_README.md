# 文档知识图谱抽取技能

自动化从 Word/PDF 文档抽取 7+ 级层次化知识节点，标注认知层级和语义关系的抽取技能。

## 🚀 快速启动

### 安装技能到 OpenCode
```bash
git clone https://github.com/flyboat403/knowledge-graph-extraction-skill.git ~/.agents/skills/knowledge-graph-extractor
```

### 使用命令
```bash
cd ~/.agents/skills/knowledge-graph-extractor
python scripts/extract_knowledge_graph.py \
  --source /path/to/your_document.docx \
  --template assets/template.xlsx \
  --output /path/to/output_result.xlsx
```

## 📊 输出规格 (A-O列)

```
A列(一级): 课程类别     →  电子信息类职业技能考试  
B列(二级): 模块分类     →    专业知识(应知) / 技能操作(应会)
C列(三级): 技术领域     →      电工技术基础与技能  
D列(四级): 知识主题     →        安全用电常识
E列(五级): 具体知识点   →          触电的种类和形式
H列(关系): 前置知识     →          电阻概念;电流概念 (先修依赖)  
I列(关系): 后置知识     →          安全措施 (后续学习)
J列(关系): 关联知识     →          静电防护;雷电防护 (相关知识点)
```

| 列 | 用途 | 约束 |
|----|------|------|
| A-G | 层级树结构 | 每行仅1个知识点 |
| H-I-J | 前置/后置/关联 | 英文`;`分隔多个知识点 |
| K | 标签 | 重点/难点/考点... |
| L | 认知维度 | 记忆/理解/应用/分析/评价/创造 |
| M | 知识分类 | 亊实性/概念性/程序性/元认知 |
| N-O | 目标/说明 | 教学目标/知识点说明文本 |

## 🎯 核心能力

| 能力 | 规范 | 说明 |
|------|------|------|
| **树状抽取** | 每行仅1知识点(A-G列) | 保证层级结构完整性 |
| **认知标注** | 6级自动识别 | 基于布鲁姆分类法 |
| **关系生成** | H-I-J三元关系 | 前置/后置/关联 |
| **反模式防护** | 内置验证机制 | 防止输入错误导致的问题 |
| **导入兼容** | Neo4j/Jena直接导入 | 符合标准图谱格式 |
