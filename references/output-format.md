## 列结构定义

| 列 | 字段名 | 用途 | 约束 |
|----|--------|------|------|
| A | 一级知识点 | 最高层级 | 课程/学科名称 |
| B | 二级知识点 | 模块层级 | 模块/单元名称 |
| C | 三级知识点 | 章节层级 | 章节/部分名称 |
| D | 四级知识点 | 主题层级 | 主题/知识点组 |
| E | 五级知识点 | 具体知识点 | 最细粒度知识点 |
| F | 六级知识点 | 扩展层级 | 可选 |
| G | 七级知识点 | 扩展层级 | 可选 |
| H | 前置知识点 | 学习前置依赖 | 多个用 `;` 分隔 |
| I | 后置知识点 | 学习后续内容 | 多个用 `;` 分隔 |
| J | 关联知识点 | 相关知识点 | 多个用 `;` 分隔 |
| K | 标签 | 重点/难点标记 | 多个用 `;` 分隔 |
| L | 认知维度 | 学习目标层次 | 单选 |
| M | 分类 | 知识类型 | 单选 |
| N | 教学目标 | 学习目标描述 | 文本 |
| O | 知识点说明 | 详细说明 | 文本 |

## 层级关系约束

### 核心规则：树状结构

**每行只能填写一个知识点**（A-G列中只有一个非空单元格）

知识点所在的列位置表示其层级：
- A列 = 一级（根节点）
- B列 = 二级（A的子节点）
- C列 = 三级（B的子节点）
- ... 以此类推

### 正确示例

```
行号  A列(一级)      B列(二级)        C列(三级)        D列(四级)    E列(五级)
3     演示文稿
4                    新建演示文稿
5                                    幻灯片大小
6                    演示文稿设计
7                                    主题
8                                                    背景
9                    增删幻灯片
10                                   新建幻灯片
```

### 错误示例（禁止）

```
❌ 错误：一行填写多个层级
['演示文稿', '新建演示文稿', '幻灯片大小', '', '', '', '']

✅ 正确：每行一个知识点
['演示文稿', '', '', '', '', '', '']
['', '新建演示文稿', '', '', '', '', '']
['', '', '幻灯片大小', '', '', '', '']
```

## 知识点关系约束

### H列：前置知识点

学习当前知识点前需要先掌握的知识点。

**识别规则：**
1. 基础概念 → 高级概念（如：电阻 → 欧姆定律）
2. 工具使用 → 应用操作（如：万用表使用 → 电流测量）
3. 理论知识 → 实践技能（如：电路原理 → 电路安装）

### I列：后置知识点

学习当前知识点后可以继续学习的知识点。

**生成规则：**
- 自动由 H列 生成：如果 A的前置是B，则 B的后置是A
- 确保关系的双向性

### J列：关联知识点

与当前知识点相关但不构成前置/后置关系的知识点。

**识别规则：**
1. 同一主题下的并列知识点
2. 相似概念或对比概念
3. 不同章节但逻辑相关的知识点

### 关系约束

1. **唯一性**：任意两个知识点之间只能存在一种关系
2. **数量上限**：关系设置总数上限 2000 个
3. **分隔符**：多个知识点用英文分号 `;` 分隔

## 标签与分类约束

### K列：标签

| 固定标签 | 说明 |
|----------|------|
| 重点 | 核心知识点 |
| 难点 | 学生普遍困难的知识点 |
| 考点 | 考试常考知识点 |
| 课程思政 | 思政教育相关知识点 |

可自定义标签，多个标签用 `;` 分隔。

### L列：认知维度（单选）

| 维度 | 说明 | 典型动词 |
|------|------|----------|
| 记忆 | 识别、回忆 | 识别、回忆、复述 |
| 理解 | 解释、举例 | 解释、举例、分类 |
| 应用 | 使用、运用 | 运用、使用、操作 |
| 分析 | 分解、辨析 | 分析、比较、归纳 |
| 评价 | 判断、论证 | 评价、判断、论证 |
| 创造 | 生成、设计 | 设计、创作、规划 |

### M列：知识点分类（单选）

| 分类 | 说明 | 示例 |
|------|------|------|
| 事实性 | 具体事实、数据 | 电压单位是伏特 |
| 概念性 | 概念、原理、定义 | 欧姆定律的定义 |
| 程序性 | 操作步骤、方法 | 万用表测量电阻的方法 |
| 元认知 | 学习策略、方法选择 | 如何选择合适的解题方法 |
