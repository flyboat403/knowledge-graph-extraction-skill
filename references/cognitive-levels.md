# 认知层级定义

基于布鲁姆教育目标分类学（Bloom's Taxonomy），定义六级认知维度。

## 认知维度层级

```
创造 (Create)      ← 最高层级
    ↑
评价 (Evaluate)
    ↑
分析 (Analyze)
    ↑
应用 (Apply)
    ↑
理解 (Understand)
    ↑
记忆 (Remember)    ← 最低层级
```

---

## 详细定义

### 1. 记忆 (Remember)

**定义**：识别、回忆事实性知识

**典型动词**：
- 识别、回忆、复述、列举
- 背诵、记忆、说出、写出

**知识点示例**：
- 说出电阻的单位
- 列举常用电阻器的种类
- 复述安全用电的基本原则

**判断依据**：
- 文档中出现"了解"、"知道"等动词
- 纯粹的事实性陈述
- 不需要理解内在逻辑

---

### 2. 理解 (Understand)

**定义**：解释、举例、分类，把握知识含义

**典型动词**：
- 解释、举例、分类、描述
- 说明、阐述、理解、区分

**知识点示例**：
- 解释电路的三种状态
- 理解电阻的概念
- 说明电功率的含义

**判断依据**：
- 文档中出现"理解"动词
- 需要解释原因或原理
- 能够用自己的话表述

---

### 3. 应用 (Apply)

**定义**：在新情境中使用知识

**典型动词**：
- 运用、使用、操作、执行
- 应用、计算、测量、分析

**知识点示例**：
- 用万用表测量电阻值
- 应用欧姆定律计算电流
- 使用示波器测量电压

**判断依据**：
- 文档中出现"掌握"、"能够"动词
- 涉及具体操作或计算
- 需要将知识应用于实际问题

---

### 4. 分析 (Analyze)

**定义**：分解、辨析关系，理解组织结构

**典型动词**：
- 分析、比较、归纳、推断
- 分解、辨析、找出、确定

**知识点示例**：
- 分析电路故障原因
- 比较不同放大电路的特点
- 归纳逻辑电路的设计规律

**判断依据**：
- 文档中出现"分析"、"比较"动词
- 需要分解复杂问题
- 找出各部分之间的关系

---

### 5. 评价 (Evaluate)

**定义**：做出判断、论证观点

**典型动词**：
- 评价、判断、论证、评估
- 选择、决定、验证、批判

**知识点示例**：
- 评价不同方案的优劣
- 判断电路设计的合理性
- 选择最适合的测量方法

**判断依据**：
- 文档中出现"评价"、"判断"动词
- 需要做出价值判断
- 基于标准进行评估

---

### 6. 创造 (Create)

**定义**：生成新观点、设计方案

**典型动词**：
- 设计、创作、规划、构建
- 开发、制定、发明、综合

**知识点示例**：
- 设计一个控制系统
- 创作一个教学方案
- 规划项目实施方案

**判断依据**：
- 文档中出现"设计"、"创新"动词
- 需要产生新的作品或方案
- 综合运用多种知识

---

## 动词到认知维度的映射

| 文档动词 | 认知维度 |
|----------|----------|
| 了解、知道、听说 | 记忆 |
| 理解、明白、清楚 | 理解 |
| 掌握、能够、会 | 应用 |
| 重点掌握 | 应用（高优先级） |
| 分析、比较、归纳 | 分析 |
| 评价、判断、选择 | 评价 |
| 设计、创作、规划 | 创造 |

---

## 代码实现

```python
def get_cognitive_level(verb: str) -> str:
    """根据动词判断认知维度"""
    verb_lower = verb.lower()
    
    if "重点掌握" in verb_lower:
        return "应用"  # 高优先级
    elif any(w in verb_lower for w in ["掌握", "能够", "会", "使用", "测量"]):
        return "应用"
    elif any(w in verb_lower for w in ["理解", "明白", "解释", "说明"]):
        return "理解"
    elif any(w in verb_lower for w in ["了解", "知道", "听说"]):
        return "记忆"
    elif any(w in verb_lower for w in ["分析", "比较", "归纳"]):
        return "分析"
    elif any(w in verb_lower for w in ["评价", "判断", "选择"]):
        return "评价"
    elif any(w in verb_lower for w in ["设计", "创作", "规划"]):
        return "创造"
    else:
        return "理解"  # 默认
```
