# 📋 文档知识图谱抽取技能 - 快速使用卡片

## ⚡ 一行命令开始

```bash
python scripts/extract_knowledge_graph.py \
  --source 输入文档.docx \
  --template assets/template.xlsx \
  --output 输出结果.xlsx
```

---

## 🎯 三大核心能力

| 能力 | 效果 | 价值 |
|------|------|------|
| **6+级知识抽取** | `A-G列`树状结构 | 一层层剖析知识点层次 |
| **语义关系识别** | `H-I-J列`三类关系 | 揭示知识点逻辑联系 |  
| **认知层级标注** | `L列`六维认知 | 标明掌握程度要求 |

---

## 📊 典型输出格式

| A | B | C | D | E | H | I | J | L | K |
|--|--|--|--|--|--|--|--|--|--|
| 电子信息考试 | 专业理论 | 电工基础 | 安全用电 | 触电形式 | 电阻概念 | 电位概念 | 电流电压 | 理解 | 考点 |
| | | | 电路分析 | 串并联 | 直流电路 | 暂态分析 | 交流等效 | 应用 | 重点 |

> **💡 约束**: 每行`A-G`列只能填写**一个知识点**（树状结构）

---

## 🚀 三步骤应用流程

### 1️⃣ 准备文档  
- ✅ Word/PDF 格式的结构化内容
- ✅ 使用标题样忎而非手动编号
- ✅ 包含明确动词（"理解"/"掌握"/"应用"）

### 2️⃣ 执行抽取  
```bash
# 真实示例:
python scripts/extract_knowledge_graph.py \
  --source 05电子信息类.docx \
  --template assets/template.xlsx \
  --output 电信知识点图谱.xlsx
```

### 3️⃣ 验证结果
- [ ] 每行只1个知识点 (A-G列)
- [ ] 关系用英文";"隔开  
- [ ] 认知维度属单选(L列/M列)
- [ ] 文件可直接导入知识图谱

## ⚠️ 核心约束 (必须遵守!)

1. **树状约束**: A-G 列每行只能填写一个节点 ✅  
2. **关系分隔符**: 多个知识点用英文分号 `;` ✅
3. **单选约束**: L/M列仅填单个值 ✅
4. **完整保留**: 不删模板列 (A-O保持完整) ✅

## 📤 知识图谱导入说明

### Neo4j 导入 (Cypher)
```cypher
LOAD CSV WITH HEADERS FROM 'file:///output.csv' AS row
WHERE row.`二级知识点` IS NOT NULL
CREATE (k:Knowledge {name: row.`二级知识点`, level: 2})
```

### 用途
- 构建学科知识图谱
- 分析教学内容覆盖度  
- 规划个性化学习路径
- 评估课程难度梯度
