**一行安装**:


**核心特性**:
- ✅ 一次性提取7级知识点层级结构
- ✅ 内置关系识别(H/I/J列自动生成)  
- ✅ 认知维度自动标注(L列六级分类)
- ✅ 遵循布鲁姆教育目标分类法
- ✅ 输出兼容Neo4j/Jena导入
- ✅ 防错机制与质量保障

**使用场景**:
- 文档知识图谱结构化分析
- 教学大纲知识点抽取
- 职业技能标准解析  
- 课程资源整合
- 知识依赖关系识别
